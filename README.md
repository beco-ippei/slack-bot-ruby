## Installation

Add this line to your application's Gemfile:

```ruby
gem 'slack-bot-slim', git: 'https://github.com/beco-ippei/slack-bot-slim.git'
```

And then execute:

    $ bundle


Create your team's token from https://my.slack.com/services/new/bot
